You can cancel seats (access codes) at any time before or during a scheduled event by using the ReadyTech Admin Portal. Below are detailed cancellation policies for each event type.

Watch the video below or continue reading to learn about our cancellation policies.

Cancelling Instructor-Led events

Includes the following event types:

  • Instructor-Led
  • Instructor Prep
  • ILT Flex

Cancel un-activated access codes by 12:00 pm, on the first day of training to avoid charges for unused seats. The 12:00 pm cancellation time is based on the timezone of the event. Cancellations made after this time take effect the next day.

For example:

You have an event set to Pacific time zone (PST), and you request to cancel an access code at 1:00 pm (PST) on the first day of training. The access code is charged for the first day of training, but not for future days.

Note that ILT Flex events cannot be cancelled after 12:00 pm first day of training.

An access code is charged regardless for days session data exists. See sessions for more information.

Cancelling Self-Paced and Workshop events

Includes the following event types:

  • Self-Paced
  • Workshop

Cancel Self-Paced and Workshop events before they start to prevent charges. You cannot cancel Self-Paced events for which vouchers exist.

You cannot cancel Self-Paced events that have started.
